On this page you can change the settings of the game instead of the defaults. If you are an adventurous type, choose the random option and you will be surprised to discover new settings that are randomly set in each round. The game settings are divided into several categories:

Food Category: You can set the amount of food in the game using the slider bar, minimum is 50 balls and maxumum is 90 balls. Default value is 70.
In addition, you can choose the color of each food type - 5 points balls, 15 points balls and 25 points balls.

Monsters Category: You can set the amount of monsters in the game using the arrows inside the input box, minimum is 1 monster and maxumum is 4 monsters. Default value is 1.

Keyboard Category: You can set the movement keyboard buttons of the Pacman to any buttons that you want. Note that all selected keys must be different from each other, You can't choose the same key for different move directions Default values are ArrowUp for up, ArrowDown for down, ArrowLeft for left and ArrowRight for right.

Time Category: You can set the duration of the game for each round. Note that the minimum time is 60 seconds, and the maximun is 9 minutes and 55 seconds. You can change the minutes and the seconds input boxes seperatly depending on your choice. Default value is 60 seconds.

Sound Category: You can set the volume of the game's background playback for each round using the slider bar. Note that the minimum volume power is 1 and the maximun is 100. Note that you can also choose the sound off checbox which will set off all the sounds in the game - background playback, loosing and winning alerts. Default value is sound on and 50 volume power.
